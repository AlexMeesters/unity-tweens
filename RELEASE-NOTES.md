I'm proud to announce the release of **Tweens 3.0.0** for Unity. This release introduces a whole new architecture with a improved Tween builder!

- New architecture with a improved Tween builder
- Support for new Tween types
- Reusable Tween builders
- Automatic Tween from and to inheritance
- Tween Inspector for analyzing and debugging Tweens
- Various bug fixes and performance improvements
